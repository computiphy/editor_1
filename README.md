# Wedding Photography AI Pipeline

Local-first, open-source AI-powered pipeline for automating wedding photography post-production.

## Features
- RAW/JPEG Ingestion
- Intelligent Culling
- AI Restoration (GFPGAN, NAFNet)
- Batch Color Grading
- Narrative Grouping
- Smart Cropping (RT-DETR)
- Album Layout Generation

## Installation
```bash
pip install -e .
```

## Usage
```bash
python main.py --config configs/default.yaml
```
